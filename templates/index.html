3桁の数字を当てるゲームです。

0から9の数字を用い、同じ数字は使いません。

○987, ○012, ×989, ×227

  1. 3桁の数字をコール(call)します。
  2. コールした数字がどの程度合っているかが返ってきます。
  3. eat: 数字と桁が合っている数が返ってきます。
  4. bite: 数字は合っているが桁が違う数が返ってきます。
  5. コールした数字が3桁すべて一致すると3eatとなり、終了。

example: 3桁の数字が094の場合

  1. call: 764 -> answer: 1 eat, 0 bite.
  2. call: 351 -> answer: 0 eat, 0 bite.
  3. call: 980 -> answer: 0 eat, 2 bite.
  4. call: 869 -> answer: 0 eat, 1 bite.
  5. call: 708 -> answer: 0 eat, 1 bite.
  6. call: 094 -> answer: 3 eat, 0 bite.
